Example: Worker Lands a New Job in Japan

When a worker gets matched to a job, the Orchestrator automatically triggers a chain of agent actions across the entire ecosystem.

1. Training Agent → Schedule language course 2. Compliance Agent → Start visa process 3. Housing Agent → Find housing near workplace 4. Health Agent → Register health insurance 5. Finance Agent → Setup salary wallet + remittance
⚡

Example: Worker's Visa Expiring in 30 Days

The Compliance Agent detects the expiry and coordinates with other agents to take proactive action.

1. Compliance Agent → Alert worker & employer 2. Job Agent → Check new opportunities if needed 3. Finance Agent → Review savings for extension costs 4. Housing Agent → Adjust lease if contract changes
💡

Example: Worker Wants to Invest Savings

The Finance Agent coordinates with other agents to provide smart investment advice based on the worker's full lifecycle data.

1. Finance Agent → Analyze savings & risk profile 2. Housing Agent → Show investment properties near demand 3. Job Agent → Predict income stability for investment 4. Orchestrator → Execute investment via Basao wallet
